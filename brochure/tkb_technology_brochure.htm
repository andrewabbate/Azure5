eLearning Content Creation/Delivery Platform 
TKB Technology is an eLearning content creation and delivery platform used by an instructor to supplement in-classroom lectures with on-line testing and eLearning materials. TKB Technology can also be used to deploy a complete virtual classroom environment.

Client/Server (DNA based) Architecture
TKB was designed for the Web and all its possibilities. Its Microsoft Distributed interNet Architecture (DNA) moves the computing responsibility off the Web Server and distributes it to all the participating computers of the application. The TKB Server has an XML based API and uses a Microsoft SQL or Access database to record all the class materials published by an instructor. Client components include the TKB Author program, TKB Manager program, and a standard web browser.  Click here to learn more about TKB's unique architecture.

Smooth Migration Path To Microsoft .NET Architecture
TKB technology is Microsoft DNA-based and therefore can be migrated to Microsoft's next generation .NET architecture efficiently and cost effectively. This will allow TKB to take advantage of .NET's presentation, component, and data technologies in order to make development of rich eLearning software components, on the Internet, even more cost effective.

Authoring, Managing, And Publishing Class Materials Is So Easy
The TKB Author program, used by an instructor to manage her classes, is a Windows native executable that is rich with features. Unlike browser-based eLearning applications, TKB Author is able to use the full processing power of the instructor's workstation in order to limit the reliance on varying Internet performance. Students use a standard Web browser to access their Class Web Pages. Click  here for a video demonstration of authoring content with TKB.

Who Can Benefit From Deploying TKB Technology?
 A college, technical training center, eLearning ASP, or high school department. Also, an eLearning company can add functionality to an existing product line by using TKB as a Web service.

The TKB architecture is dynamic in that there are no pre-made HTML pages. The TKB database stores all class materials, that are delivered and rendered to an instructor, student, administrator, or service on-the-fly.
